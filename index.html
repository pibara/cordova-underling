Cordova Underling

Apache Cordova (formaly known as PhoneGap is a framework that allows you to write mobile applications for Android, IOS, etc using HTML/CSS and JavaScript. This is great, but if you are a JavaScript developer who wants to check out Cordova, you will need to do quite a bit of work before you can start playing around with the Apache Cordova JavaScript APIs. Lets take Android as an example to see what you will need to do to 'start' playing with the Cordova API using your Android phone:

  1. Install the Eclipse IDE
  2. Install the Android SDK
  3. Install the ADT plugin for Eclipse
  4. Download Apache Cordova
  5. Create an Android project in Eclipse.
  6. Move some Cordova files to your android project directory tree.
  7. Write some HTML/JavaScript that uses the Cordova API
  8. Build an APK
  9. Deploy the APK to your phone
  10. Run the application on your phone.
  11. If you need to tune anything, change your files and repeat steps 8/9/10.

Cordova Underling is a simple Cordova App that makes your first encounter with the Cordova API a much less time consuming process. If you will want to write real applications you shall need to take all steps above, but while still just exploring the Cordova API, Underling will help you get started a bit faster.

  1. Install Cordova Underling on your phone from Android Play
  2. Write some HTML/JavaScript that uses the Cordova API
  3. Upload your HTML/Javascript to a web server.
  4. Start Underling and point it to your Apps web URL.
  5. Press the Underling button to start your Cordova App.
  6. If you need to tune anything, change your files and repeat steps 3 and 5.

Basically this should help you get started playing around with Cordova faster. Maybe you find out that Cordova isn't what you need, and you won't have had to spent hours getting the Android development environment up and running. Maybe you find out Cordova is just right for you, and you will only have to install the development environment as last step (for creating and publishing your App) after you've completed and tested your App using Underling.
